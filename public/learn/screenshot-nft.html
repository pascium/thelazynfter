But before that, let's first start with the basics and explain…

What is an nft screenshot?

A screenshot is a digital representation of what someone sees on their computer screen. 

Screenshots are often taken while playing video games, watching videos, browsing websites, or using software applications. 

When you take a screenshot, you capture the image displayed on your monitor onto a file called a screenshot. 

You can then save the screenshot to your hard drive or upload it to a website where others can view it.

Screenshots are often saved to a file called a.png (Portable Network Graphics) or.jpeg (Joint Photographic Experts Group).

Now that we know what a screenshot is, it’s time to figure out….

How to screenshot an nft.

There are two ways to take screenshots. You can either use a third party extension or you can use the native OS feature.

Taking Screenshots in Firefox

To take a screenshot in Firefox, you need to navigate to the page you want to take a screenshot of and right-click on that page.

After right clicking you need to select from the popup menu the option ”take screenshot”.

Now you can select what screenshot you want, you can take a screenshot of the visible part of the page, of the whole page or use your mouse to select a specific part on the webpage.

Taking Screenshots in Android.

If you'd rather not use any third-party apps, you can simply press the power button and the volume buttons at the same time.

A popup screen should appear and allow you to select what you want to capture. Tap on the image icon and you're done!

Where is your screenshot stored?

Screenshots are saved in the following location:

  1.  Windows 10: %USERPROFILE%\Pictures\Screenshots
  2.  Mac OS X: ~/Library/Application Support/Apple/SharedSupport/ScreenshotCapture
  3.  Linux: ~/.local/share/applications/screenshotcapture.desktop

If you want to find out where a screenshot was taken, open the file manager (Windows Explorer, Finder, etc.) and navigate to the directory above. 

You should see a folder named Screenshots inside the Pictures folder. Inside this folder, you'll find the screenshots you took.

Best practices for using screenshot NFTs.

No matter what screenshot you use, always and alway make sure that you have the proper permissions or rights to use the screenshot or any other images for that matter.

If you write articles like me, and want to add some images to spice up your blogpost, you can use screenshots but you need to add the proper credit of the one that made the original image as a figcaption including a link back to the original source.

But if you are unsure if you can use a screenshot of a webpage, NFT or any other image, just send the original owner an email and ask for permission.

Why would you buy an NFT when you can screenshot it?

NFT represents ownership of a real estate property. If you own a house, you have a legal claim to the land where the house sits. 

That land could be represented by an NFT. In the same way, if you own a car, you have a legal right to drive it anywhere in the country. 

That vehicle could be represented by an ERC20 token.

When you purchase an NFT, you're essentially buying a virtual licence to use it how you see fit.

Can you get sued for Screenshotting an NFT?

Yes! You can be sued for screenshotting an NFT (Non-Fungible Token). However, if you do not have any intent to sell the token, then you should be fine. 

If you intend to resell the tokens, then you may need to make sure that you don't violate any terms of service agreements.

Can I sell someone else's art as NFT?

Yes! You can sell someone else's art (or anything) as non-fungible tokens. You just need to make sure that they have given their consent to do so. 

In order to do so, you would first need to create a contract between yourself and the person who owns the artwork. 

Once you've done so, you could then use the ERC-777 standard to transfer ownership of the token to the buyer.

Selling art without proper licence, consent or using copyrighted characters is a copyright infringement and can lead to huge fines.

Do I need to copyright NFT?

You do not need to copyright your NFT's. However, if you want to protect your work, then you should copyright them. 

Copyrighting your work helps to ensure that no one else uses your design without permission.

Do I own an NFT if I mint it?

The short answer is yes. You do own an NFT (Non-Fungible Token) even if you don't intend to use it. 

Minting an NFT does not make it any less of an asset. In fact, it makes it more valuable.

How should I handle screenshots

As a screenshot artist, you should always give proper attribution to the original creator. 

This means crediting the person who created the image, whether it was a video game, movie, book, etc. If you don't, the copyright holder has the right to take legal action against you.

  • Always ask permission before taking a screenshot of anything that belongs to someone else.

  • Use screenshots to document progress.
  • Never sell or giveaway screenshots.
